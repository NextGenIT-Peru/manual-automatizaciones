# AMI backup

Para solicitar una adición o quitar a un usario de un grupo específico, debes diligenciar los siguientes campos:

<div class="column">
  <h3>Campos para Añadir / Quitar grupos: </h3>
  <ul class="sublist">
    <li>
      <strong>Template</strong>: Requerimientos y accesos
    </li>
    <li>
      <strong>Requester</strong>: Nombre del solicitante
    </li>
    <li>
      <strong>Tipo de Requerimiento</strong>: Gestión de Usuarios
    </li>
    <li>
      <strong>Gestión de Usuarios</strong>: Gestión de grupos
    </li>
    <li>
      <strong>Grupo</strong>: Aplicaciones
    </li>
    <li>
      <strong>Aplicativo</strong>: Elegir el grupo al cual desea añadir o quitar
    </li>
    <li>
      <strong>Tipo de Carga</strong>: Individual
    </li>
    <li>
      <strong>Tipo de acción</strong>: Asignación/
    </li>
    <li>
      <strong>Grupo AD</strong>: Elegir el grupo AD específico
    </li>
    <li>
      <strong>Acciones</strong>: Agregar/
    </li>
    <li>
      <strong>País</strong>: Elegir entre los permitidos
    </li>
  </ul>
</div>

Una vez el ticket es creado, debe pasar por revisión de controller y él aprueba el ticket
para ser resuelto y elige el grupo de resolutor para que la automatización inicie.
